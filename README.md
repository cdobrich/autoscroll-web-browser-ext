# autoscroll-web-browser-ext

A simple multi-directional auto-scrolling (often called "autoscroll" or "middle-click scroll") web browser extension, Manifest V3 compatible.

## Loading Locally

How to load and test this extension:

1. Download a zip-archive of the project, and extract into a single folder on your computer.

2. Open Google Chrome and navigate to `chrome://extensions/` (this works for Brave Browser also, though Brave uses `brave://extensions/`).

3. Toggle the `Developer mode` slide switch in the top-right corner to On.

4. Click Load unpacked in the top-left corner.

5. Select the folder containing the project files.

6. Reload any existing page in your web browser before testing, (like a long news article or documentation page) and test it by middle-clicking! (Note: Extensions will not run on standard chrome:// system pages or the Chrome Web Store).

# Configuring Extension

1. Click on the Plugins toolbar button.
2. Select the three-dots menu
3. Click the `Options` item
